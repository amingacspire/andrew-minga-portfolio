Internet Speed Test

Measures download, upload, latency, jitter, and packet loss against Cloudflare's edge network using their open-source measurement engine. Takes about 30 seconds.

Starting…

Download — Mbps
Upload — Mbps
Latency (idle) — ms
Jitter — ms
Latency under load — ms
Packet loss — %

Results measure the path between your device and the nearest Cloudflare data center, so they reflect your connection quality, not a specific ISP plan number. Wi-Fi, VPNs, and busy networks all lower results; test wired if you are chasing a real number. The measurement engine is Cloudflare's open-source library; completed measurements are collected by Cloudflare for aggregate internet-quality statistics. This site itself logs nothing.
