Hash & Checksum

Compute SHA-256, SHA-1, SHA-384, and SHA-512 for a file or pasted text — to verify a vendor download matches its published checksum, or to fingerprint a config.

Hashing uses the browser's Web Crypto API and runs entirely on your device; the file is read locally and never uploaded. Case-insensitive comparison against a published checksum confirms integrity.
